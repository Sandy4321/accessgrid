  GetResourceList  
GetResourceList ()

This method reads a file generated by vic and a tcl script (courtesy Bob Olson) which contains a vic-compatible description of video capture devices on the local machine. Note: the name of the file is hardcoded Note: for now, the file should be generated at installation, and the user should be provided with instructions for generating the file by hand. later, users should be able to force generation of the file from within the AG software

An example of the file is as follows:

device: o100vc.dll - Osprey Capture Card 2 portnames: external-in device: Microsoft WDM Image Capture (Win32) portnames: external-in device: o100vc.dll - Osprey Capture Card 1 portnames: external-in
